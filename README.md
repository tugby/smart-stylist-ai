**About the Project**

This application is an AI-powered smart stylist that recommends the most similar H&M clothing items based on user input, using NLP techniques.

**Check out the live demo here:**

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://smart-stylist-ai.streamlit.app/)

**🚀 Technologies**

* **Python**
* **Streamlit** (UI)
* **Sentence-Transformers** (AI Model)
* **Pandas** (Data Processing)

**📊 Dataset**

The H&M dataset used in this project is sourced from Kaggle.
* [Dataset Source](https://www.kaggle.com/c/h-and-m-personalized-fashion-recommendations)
